### 部署运行
#### 宝塔面板部署（正式发布推荐）
1. 安装宝塔面板
参考官方文档安装宝塔面板：https://www.bt.cn/
2. 下载代码，上传代码，添加站点并配置。
[点击这里，查看详细部署教程，全景式，沉浸式教你如果运行起自己的likeshop](http://doc.likemarket.net/likeshop_develop_v2_x/2107074) 

#### phpStudy部署（本地部署推荐）
1. 下载安装phpStudy（建议8.1版本）：https://www.xp.cn/download.html
2. 安装完以后，打开phpStudy，启动mysql和nginx。
3. 下载代码，添加站点并配置。
[点击这里，查看详细部署教程，全景式，沉浸式教你如果运行起自己的likeshop](http://doc.likemarket.net/likeshop_develop_v2_x/2119296) 

#### docker部署
1. 下载自己系统适合的安装包，安装docker。docker官方下载地址：[下载地址](https://www.docker.com/get-started)
2. 启动docker软件，设置docker镜像源。
3. 进入likeshop/docker目录，运行` docker-compose -p likeshop down &amp;&amp; docker-compose -p likeshop up -d` （调试时时，可去掉-d参数，-d参数是让docker后台运行），一键快速启动docker。
[点击这里，查看详细部署教程，全景式，沉浸式教你如果运行起自己的likeshop](http://doc.likemarket.net/likeshop_develop_v2_x/2107075)
